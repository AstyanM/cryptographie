	{(Description)}
	  °°°°°°°°°°°


This Python package aims to access various cryptographic modules which take into account :

- Caesar's encryption
	1. Encrypt
	2. Decrypt with the key
	3. Decrypt without the key

- The Transposition method
	1. Encrypt
	2. Decrypt with the key
	3. Decrypt without the key

- Encryption by Substitution
	1. Encrypt
	2. Decrypt with the key
	3. Decrypt without the key

- The Vigenere cipher
	1. Encrypt
	2. Decrypt with the key

- Hash's method :
	* SHA-256
	* SHA-512
	* SHA3-512
	* MD5
	* BLAKE

	1. Encrypt
	2. Decrypt without the key

- One-time Pad encryption
	1. Encrypt
	2. Decrypt with the key

- RSA encryption
	1. Encrypt
	2. Decrypt with the key

This project includes two dictionaries downloaded from the Internet. They are accessible by Dicts, Python/Dicts or Executable/Dicts. They can therefore be modified to
be optimized (add common words, first names, etc.) or you can add them to take into account other languages, but the program must therefore be modified.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

	{(Launch)}
	  °°°°°°


If you are using Python 3.x:
for the first launch of the program, please first launch INSTALLER.bat, which aims to install the various modules necessary for the correct functioning of the program. 
Then you can launch it from Launcher.bat, which will run Python/main.py

If you are not using a version of Python that matches 3.x or if Python is not installed: 
you can update it by downloading the latest version of Python from the official website: https://www.python.org/downloads/
To see which version you are using, use the "python -V" command in the cmd.

If you don't want to use python:
you can run the program compiled using pyinstaller from Launcher.bat, which will run Executable/main.exe
